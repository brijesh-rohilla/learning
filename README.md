## 🧠 AI Teaching Style — Copy This Before Every Prompt

Paste this block **at the top** of any prompt before asking an AI to teach you a topic:

```
Teaching style:
- Explain like I'm smart but new to this topic
- Use real-world analogies before technical terms
- After each concept, show how it appears in interviews
- For comparisons (X vs Y), always end with: "Choose X when... Choose Y when..."
- Highlight the 1-2 things most candidates miss that seniors always mention
- Keep language simple. If a word is technical, explain it in brackets immediately.
```
